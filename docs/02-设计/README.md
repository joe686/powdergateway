# 02-设计

技术设计目录。分两层：**总体架构** + **单元详细设计**。

## 总体架构

| 文件 | 说明 |
|------|------|
| [架构说明.md](./架构说明.md) | 模块拆分、目录结构、菜单树、数据库表设计、跨模块依赖 |
| [技术栈总结.md](./技术栈总结.md) | 完整技术栈一览（后端/前端/数据库/缓存/中间件版本） |
| [实现范式.md](./实现范式.md) | 后端 Controller/Service/AOP、前端页面/请求/状态等核心代码模式 |

## 单元详细设计

[详细设计/](./详细设计/) 下按 `日期-单元-design.md` 命名，每份对应一个交付单元的技术方案（数据结构、接口契约、关键算法、风险点）。命名约定：

```
YYYY-MM-DD-{单元编号}-{特征词}-design.md
例：2026-05-13-sys3-user-permission-design.md
```
