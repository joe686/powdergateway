## 阅读效率约定

**为节约上下文与人力开支：**

1. **先读本索引 + 根 CLAUDE.md**，不到需要不深入子目录
2. **反馈处理**用 [pg-feedback-flow](../.claude/skills/pg-feedback-flow/SKILL.md) skill 走固定流程，避免每次问"应该记到哪"
3. **变更登记**用 [变更记录.md](03-开发/变更记录.md) 顶部的模板，不要每次重造格式
4. **任务分解**放 `docs/03-开发/任务计划/` 下按日期命名，实施完可清（保留 CHG 引用即可追溯）
5. **不重复登记**：一件事只在一处记详细版本，其他地方用链接引用；避免两处漂移
