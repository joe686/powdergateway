## 4. 类结构

### 工具类

**`utils/AesUtil.java`**
- `encrypt(String plaintext, String key): String` — AES-128/ECB/PKCS5Padding，Base64 编码输出
- `decrypt(String ciphertext, String key): String` — 逆向解密
- key 注入自 `${powergateway.aes.key}`

### 业务层

**`service/DbConnectionService.java`**

| 方法 | 说明 |
|------|------|
| `list(): List<DbConnectionVO>` | 查全量，password 替换为 `***` |
| `save(DbConnectionSaveRequest): Long` | 新建或更新；密码传 `***` 时不覆盖原值；AES 加密后落库；注册/刷新动态数据源 |
| `delete(Long id): void` | 软删除；从 DynamicRoutingDataSource 移除 |
| `testConnection(Long id): TestResult` | 解密密码 → `DriverManager.getConnection()` 3秒超时 |
| `registerDataSource(DbConnection conn): void` | 私有方法，封装注册逻辑，供 save 和 Initializer 复用 |

### Controller

**`controller/DbConnectionController.java`**

| 方法 | HTTP | 路径 |
|------|------|------|
| `list` | GET | `/api/db/list` |
| `save` | POST | `/api/db/save` |
| `delete` | DELETE | `/api/db/{id}` |
| `testConnection` | POST | `/api/db/{id}/test` |

### 启动初始化

**`config/DbConnectionInitializer.java`**
实现 `ApplicationRunner`，`run()` 方法：加载全量未删除连接 → 逐条调用 `registerDataSource()`。

## 5. 数据源 Key 约定

动态数据源 key 统一为 `"db_" + dbId`（如 `db_1`、`db_42`）。
后续 M2-2 及以后单元通过此 key 切换目标库执行 SQL。

## 6. 接口行为规范

### GET /api/db/list
- 返回全部 `deleted=0` 的连接
- `password` 字段固定返回 `"***"`

### POST /api/db/save
- `id` 为 null → 新建；`id` 不为 null → 更新
- 密码字段传 `"***"` → 不更新原密码（编辑时保持原值）
- 成功后立即向 `DynamicRoutingDataSource` 注册或刷新该数据源
- 返回连接 `id`

### DELETE /api/db/{id}
- MyBatis-Plus 逻辑删除（`deleted=1`）
- 从 `DynamicRoutingDataSource` 移除对应 key

### POST /api/db/{id}/test
- 从 DB 加载连接配置，AES 解密密码
- 按 `dbType` 确认 JDBC Driver（MySQL/Oracle/PostgreSQL）
- `DriverManager.getConnection(url, user, password)` 超时 3 秒（通过 Properties 设置 `connectTimeout=3000`）
- 成功：`{"success": true, "message": "连接成功，耗时 xxms"}`
- 失败：`{"success": false, "message": "具体异常信息"}`

## 7. DTO 定义

### DbConnectionSaveRequest
```
id         Long      可选，更新时传入
name       String    非空
dbType     String    MySQL / Oracle / PostgreSQL
url        String    非空
username   String    非空
password   String    新建必填；编辑传 "***" 表示不变
env        String    dev / test / prod
poolSize   Integer   可选，默认 5
timeout    Integer   可选，默认 3000（毫秒）
```

### DbConnectionVO（返回视图对象）
- 字段与 DbConnection 相同，password 固定为 `"***"`

## 9. 前端设计

**文件**：`frontend/src/views/db/ConnectionList.vue`
**菜单路径**：`/db/connections`（「可视化接口开发 → 数据库连接管理」）

### 布局
- 顶部：「新建连接」按钮（右对齐）
- 主体：`el-table`，列：连接名 | 类型 | 环境 | URL | 创建时间 | 操作

### 操作列
- 「测试」：调 `POST /api/db/{id}/test`，绿/红 `el-message` 反馈
- 「编辑」：打开同一弹窗，密码显示为 `***`，不修改则原值保持
- 「删除」：`el-popconfirm` 二次确认后调 `DELETE /api/db/{id}`

### 新建/编辑弹窗表单字段
| 字段 | 组件 | 校验 |
|------|------|------|
| 连接名 | el-input | 必填 |
| 数据库类型 | el-select（MySQL/Oracle/PostgreSQL）| 必填 |
| JDBC URL | el-input | 必填 |
| 用户名 | el-input | 必填 |
| 密码 | el-input（password 类型）| 新建必填 |
| 环境 | el-select（dev/test/prod）| 必填 |

### API 文件
`frontend/src/api/db.js`：封装 list、save、delete、test 四个请求。

## 10. 验收标准

1. 保存 MySQL 连接后，调用测试接口返回成功
2. 数据库中 `password` 字段为 AES 加密密文，非明文
3. 接口返回的 `password` 字段为 `"***"`
4. 删除连接后，`DynamicRoutingDataSource` 中该 key 已不存在
5. 重启应用后，已有连接自动重新注册到动态数据源
6. 全部测试通过（`mvn test`）
