# PowerGateway 开发计划

> 基于《需求拆分与最小实现方案.md》，共 27 个交付单元，分 5 个阶段推进。

---

## 阶段一：基础脚手架（P0，5单元）

| 单元 | 内容 | 验收标准 |
|------|------|---------|
| **P0-1** | Spring Boot 骨架：pom.xml 引入全依赖、统一响应体 `Result<T>`、全局异常处理 | `/swagger-ui.html` 能访问 |
| **P0-2** | Vue3 骨架：路由+Pinia+axios封装、基础布局（侧边菜单+顶栏+内容区） | `localhost:5173` 显示登录框架 |
| **P0-3** | 配置库建表 SQL：8张表 DDL + MyBatis-Plus 实体类 | `init.sql` 执行无报错 |
| **P0-4** | 用户登录鉴权：Sa-Token + 路由守卫，预置 `admin/Admin@123` | 未登录跳转登录页 |

## 阶段二：报文格式转换模块（M1，6单元）

| 单元 | 内容 | 关键技术 |
|------|------|---------|
| **M1-1** | 格式转换引擎（后端工具类）：JSON↔XML↔CSV↔FormData 12种组合；**前端** `FormatConvert.vue`：直接转换 + 模板选择（调用 M1-6 全流程接口） | Jackson/Dom4j/OpenCSV |
| **M1-3** | 字段加工规则引擎（策略模式）：截位/补位/大小写/去空格/类型转换，多规则叠加 | `FieldProcessStrategy` 接口 |
| **M1-2** | 字段映射配置：拖拽左右树、固定值填入、自动匹配、保存模板 | `vue-draggable-next`+`el-tree` |
| **M1-5** | 转换模板管理 CRUD：分页搜索、复制、版本留存（旧版本 is_latest=0） | MyBatis-Plus 逻辑删除 |
| **M1-4** | 渠道模板管理：渠道码绑定识别字段+模板，运行时自动路由；**渠道级别报文头默认配置**（CHG-002） | Redis 缓存渠道配置 |
| **M1-6** | 报文转换调用接口：`POST /api/convert`，串联全流程，模板 Redis 缓存 TTL 10min；**CHG-004**：`applyMapping` 支持点号路径访问 XML 嵌套字段 | `@Cacheable` |
| **M1-7** ✅ | 端口分发路由：`port_route` 表 + `POST /api/dispatch`，双向转换（请求加工→转发→应答加工→返回），RestTemplate 转发，端口连通测试；**前端**：`views/convert/PortRoute.vue`，菜单路径 `/convert/port-route`；**CHG-002**：两级报文头配置（渠道默认+路由覆盖）、`HeaderConfigMerger`、`CharsetConverter`（字节级转码） | `RestTemplate` |

## 阶段三：可视化接口开发核心（M2前段，8单元）

| 单元 | 内容 | 关键点 |
|------|------|--------|
| **M2-1** | 数据库连接管理：MySQL/Oracle/PG，密码 AES-128 加密，测试连通 | `dynamic-datasource` 动态切换 |
| **M2-2** | 表结构查询：`DatabaseMetaData` + Redis 缓存（TTL 24h）+ 手动刷新 + Excel 导出 | 其他单元复用此缓存 |
| **M2-9** | SQL 审计日志：AOP 拦截 Executor → 异步队列写审计库，含 before_snapshot | `LinkedBlockingQueue + @Async` |
| **M2-3** | 查询接口配置：4步向导（选表→选字段→配置条件→预览），支持多表 LEFT JOIN | `QueryBuilder.java` |
| **M2-4** ✅ | 插入接口配置：多表（≤3张）事务、3种数据来源（请求字段/固定值/计算值） | `DataSourceResolver`（后续复用） |
| **M2-5** | 修改接口配置：强制唯一条件校验、修改前快照，复用 M2-4 的 DataSourceResolver | `UpdateBuilder.java` |
| **M2-6** | 删除接口配置：待删数据预览、批量删除保护开关 | `DeleteBuilder.java` |
| **M2-7** | 接口发布：状态流转 draft→published→disabled，统一执行入口 `/api/exec/{id}`，Swagger 动态注册 | `OpenApiCustomizer` |

## 跨单元复用规约

开发时必须复用以下组件，**不得重复实现**：

| 组件/类 | 实现单元 | 被复用方 |
|---------|---------|---------|
| `FieldProcessor`（字段加工引擎，策略模式） | M1-3 | M2-3/4/5 |
| `DataSourceResolver`（请求字段/固定值/计算值解析） | M2-4 | M2-5 |
| `ColumnValidator`（基于表结构元数据的字段校验） | M2-4 | M2-5 |
| `DatabaseMetaData` 表结构查询（Redis缓存） | M2-2 | M2-3/4/5/6 |
| `sys_config` KV 配置读取 | SYS-4 | M2-9、M2-10、SYS-1 |
| 条件配置前端组件 `ConditionBuilder.vue` | M2-3 | M2-5、M2-6 |

---

## 关键约束

- **性能目标**：查询 ≤300ms，增删改 ≤500ms
- **三库分离**：配置库 / 业务库（动态连接）/ 审计库（独立 MySQL）
- **密码安全**：登录密码 BCrypt，数据源密码 AES-128
- **不重复造轮子**：上表中的复用组件必须跨单元共用，不允许各单元各自实现
